# Item Purchase Tool

## Описание проекта

**Item Purchase Tool** — это Salesforce-приложение для управления товарами, оформления покупок и контроля складских остатков.

Приложение позволяет пользователям просматривать доступные товары, добавлять их в корзину, оформлять покупки и автоматически обновлять количество товаров на складе.

Для менеджеров реализована возможность создания новых товаров, а также автоматические уведомления при отсутствии товара на складе.

Проект разработан с использованием:
- Lightning Web Components (LWC)
- Apex Classes
- Record-Triggered Flows
- Custom Settings
- Salesforce Custom Notifications

# Функциональность

## Каталог товаров

Пользователь может:

- просматривать список доступных товаров;
- фильтровать товары по категории и типу;
- выполнять поиск по названию;
- просматривать подробную информацию о товаре;
- добавлять товары в корзину.


## Корзина и оформление покупки

Реализован процесс оформления покупки:

1. Пользователь добавляет товары в корзину.
2. При оформлении заказа создаётся запись `Purchase__c`.
3. Для каждого товара создаётся запись `PurchaseLine__c`.
4. Количество товара на складе автоматически уменьшается.
5. Пользователь перенаправляется на созданную запись покупки.


## Управление товарами для менеджеров

Для пользователей с правами менеджера доступна кнопка:

`Create Item`

Она позволяет создавать новые товары через интерфейс LWC.


## Интеграция Unsplash API

Для автоматического заполнения изображений товаров реализована интеграция с Unsplash API.

При создании товара:

1. Используется название товара как поисковый запрос.
2. Выполняется запрос к Unsplash API.
3. Полученная ссылка на изображение сохраняется в поле: Image__c

# Salesforce Objects

## Item__c

Объект товаров.

Основные поля:

| Поле | Описание |
|---|---|
| Name | Название товара |
| Description__c | Описание |
| Family__c | Категория |
| Type__c | Тип товара |
| Price__c | Цена |
| AvailableQuantity__c | Количество на складе |
| Image__c | Ссылка на изображение |


---

## Purchase__c

Объект покупки.

Поля:

| Поле | Описание |
|---|---|
| ClientId__c | Клиент |
| GrandTotal__c | Общая стоимость |
| TotalItems__c | Количество товаров |


---

## PurchaseLine__c

Строки покупки.

Поля:

| Поле | Описание |
|---|---|
| PurchaseId__c | Связанная покупка |
| ItemId__c | Товар |
| Amount__c | Количество |
| UnitCost__c | Цена единицы товара |

# Automation

## Out Of Stock Notification Flow

Создан Record-Triggered Flow для объекта `Item__c`.

Flow запускается после изменения количества товара.

Условия: AvailableQuantity__c = 0

После выполнения:

- получает получателей уведомлений из Custom Setting;
- отправляет Salesforce Bell Notification;
- отправляет Email Notification.


---

# Custom Setting

Создан Hierarchy Custom Setting: Inventory Notification Settings

Используется для хранения получателей уведомлений.

Поле: Recipient_Email__c


---

# Custom Notification

Создан тип уведомления: Item Out Of Stock

Используется Flow для отправки уведомления менеджерам.

# User Stories

## US-01 Создание покупки

**Как пользователь**

Я хочу выбирать товары и оформлять покупку,

чтобы создавать записи заказов клиентов.


Критерии:

- товары отображаются;
- товары добавляются в корзину;
- создаётся Purchase;
- уменьшается складской остаток.


---

## US-02 Управление товарами

**Как менеджер**

Я хочу создавать новые товары,

чтобы поддерживать актуальный каталог.


Критерии:

- доступна кнопка создания товара;
- новый товар сохраняется;
- изображение загружается автоматически.


---

## US-03 Контроль остатков

**Как менеджер склада**

Я хочу получать уведомление при отсутствии товара,

чтобы вовремя пополнять склад.


Критерии:

- Flow запускается при остатке 0;
- отправляется Bell Notification;
- отправляется Email.
